# Previewer

## Start previewer

```bash
npm install
npm run previewer
```

## Build previewer

```bash
npm run previewer:build
```

## Use your own resources

1. Open `http://localhost:5173` after starting previewer.
2. Paste resource URLs in the `sources` textarea (one per line).
3. Adjust all controls (this previewer covers all `RandomLoadingIndicatorProps`).
4. Read each parameter block in the controls panel for type/default/details.
5. Copy generated code from `Generated Code`.
